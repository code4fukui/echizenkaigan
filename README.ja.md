# 越前海岸盛り上げ隊マップ

越前海岸の魅力的な観光スポットを紹介するWebアプリケーションです。CSV形式のデータからマップ上にスポットを表示し、それぞれの施設情報を確認することができます。

## デモ
https://code4fukui.github.io/echizenkaigan/

## 機能
- CSV形式のデータを読み込み、マップ上にスポットを表示
- スポットをクリックすると施設の詳細情報を表示
- 越前海岸の観光情報を知ることができる

## 必要環境
特に必要なものはありません。Webブラウザがあれば利用できます。

## 使い方
Webブラウザでデモページを開くと、自動的にマップが表示されます。スポットをクリックすると、施設の詳細情報を確認できます。

## データ・API
越前海岸盛り上げ隊が提供するCSVデータを使用しています。

## ライセンス
特に指定がありません。
